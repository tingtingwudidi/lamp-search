<a name="chinese"></a>
## 🇨🇳 中文说明

一个专为新加坡三宝佛光寺设计的轻量级、直观的灯位搜索 Web 应用。旨在帮助信众通过编号快速定位在寺庙各区域的供灯位置。

### ✨ 系统特色
* **精准定位**：支持根据编号自动解析区域、排数和具体位置。
* **多区域覆盖**：包含走廊（70排）、大殿前（7排）、后殿前（39排）及四面佛大堂（39排）。
* **智能引导**：针对不同区域提供人性化的起始点方位指引（如：“面朝大门，您的右侧为首排”）。
* **禅意设计**：采用朱红与古金调性，营造庄严静谧的视觉体验。

### 🛠️ 技术实现
* **前端**: HTML5, CSS3, Vanilla JavaScript (无第三方依赖)。
* **适配**: 完美支持移动端。

<a name="english"></a>
## 🇺🇸 English Description

A custom-built, lightweight web application designed for the **Lord Buddha Temple in Singapore**. This system helps devotees quickly locate their offered lamps across various temple halls based on lamp ID numbers.

### ✨ Key Features
* **Precise Localization**: Automatically parses IDs to determine the specific area, row number, and position.
* **Comprehensive Coverage**: Covers the Corridor (70 rows), Main Hall (7 rows), Rear Hall (39 rows), and Circle Hall (39 rows).
* **Smart Navigation**: Provides human-friendly orientation cues (e.g., "Facing the entrance, Row 1 begins on your right-hand side").
* **Sacred Aesthetic**: A visually serene design using traditional crimson and gold color palettes.

### 🛠️ Tech Stack
* **Frontend**: HTML5, CSS3, Vanilla JavaScript (No external libraries required).
* **Responsive**: Fully optimized for mobile devices.
